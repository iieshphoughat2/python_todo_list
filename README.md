# Python To-Do List 

A simple to-do list application built using Python.

## Features
- Add tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Error handling

## Tech Used
- Python (loops, lists, dictionaries, error handling, conditionals)

## How to Run
1. Clone the repository
2. Run the Python file:
   python todo_list.py
